# gruntProject
形成一个可开发的打包项目
目录结构
- -gruntProject

      |-demo  开发前的代码写入点

      |-dist  开发后自动生成

      |-lib 单元测试源文件

      |-node_modules node相关的工具代码

      |-spec 测试文件夹

          |-helpers 测试自定义的matcher

          |-support 测试所需要的配置文件

          |-jasmine_examples 测试文件所在的位置

      |-bowerJs bower通用文件夹

      |-static 静态文件存放

使用的测试框架为jasmine
